# Stacks

Stacks hold language and ecosystem knowledge shared by multiple roles and
surfaces.

Examples:

- `js-ts` - JavaScript/TypeScript, Node.js, package managers, linting, testing,
  application architecture, and verification mapping.

Do not put product-specific commands or environment values here. Use repo-local
overlays for exact scripts and targets.

Stack catalogs may list conditional framework references for discovery, but
loading a stack must not imply that every listed framework is present.
