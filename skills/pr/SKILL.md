# PR Skill

Use for scoped PR work.

1. Confirm current repo and branch.
2. Inspect package scripts and docs.
3. Run local validation.
4. Self-review diff.
5. Commit intentionally.
6. Push and create/update PR.
7. Watch CI.
8. Address review threads in-thread.
9. Verify no unresolved comments remain.
